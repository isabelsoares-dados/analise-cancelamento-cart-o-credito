# analise-cancelamento-cartao-credito
Credit card cancellation analysis — identifying the main reasons why customers cancel their cards

## PROBLEMA:
Você trabalha em uma grande empresa de Cartão de Crédito e o diretor da empresa percebeu que o número de clientes que cancelam seus cartões tem aumentado significativamente, causando prejuízos enormes para a empresa.
O que fazer para evitar isso? Como saber as pessoas que têm maior tendência a cancelar o cartão?

## DATASET:
Uma base de dados com as informações dos clientes, tanto clientes atuais quanto dos clientes que cancelaram o cartão.
Banco de Dados retirado na plataforma Kaggle, link de acesso: <https://www.kaggle.com/datasets/sakshigoyal7/credit-card-customers>

## FERRAMENTAS UTILIZADAS:
A análise de dados foi realizada utilizando a linguagem Python. Usei as bibliotecas Pandas e Plotly para visualizar os dados, construir gráficos e 
com isso investigar por que os clientes cancelaram os cartões de crédito.

## INSIGHTS:
Foram 3 principais insights descobertos a partir dessa análise: 

**1.** clientes com mais produtos, tem menos taxa de cancelamento;

**2.** clientes que usam mais o cartão cancelam menos;

**3.** clientes que acionaram mais o suporte têm maior tendência ao cancelamento;
